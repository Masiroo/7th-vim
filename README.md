## Install

```bash
# to home folder
cd ~

# download .vimrc file
curl -L https://raw.githubusercontent.com/dofy/vimrc/master/.vimrc > ~/.vimrc

# install
curl -L https://raw.githubusercontent.com/dofy/vimrc/master/install.sh | sh
```
